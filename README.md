# Gulu UI for Vue 3

## 安装

```bash
yarn add gulu-ui-myself
```

或

```bash
npm intsall gulu-ui-myself
```

## 版本说明

### 0.0.1

添加了`Switch` `Button` `Tabs` `Dialog`组件功能

### 0.0.2

对已有的功能进行优化，及添加并完善功能。具体如下：

1. `Switch`组件

- 增加`style`属性，实现自定义颜色样式
- 增加`size`属性，实现尺寸选择

2. `Button`组件

- `level`增加了更多属性值：`primary` `success` `info` `warning` `danger`
